# img-lazy-processor

> Through the modern browser Observer API of Mutation Observer and Intersection Observer, lazy loading of img tags and friendly prompts for empty addresses or loading failures

## Usage

### Lint File
```bash
yarn lint
```

### Test File
```bash
yarn test
```

### Build File
```bash
yarn build
```
